==========================================================================
Description
==========================================================================
This extension integrate Live Chat (like Zopim, LiveZilla, etc) on your site and works for code of "floating button" and other on-site overlay window.

To integrate Live Chat system on your site, please follow the steps below: 
==========================================================================
Installation
==========================================================================

for Opencart 2.x.x.x
- First You should to backup your opencart site and also database.
- Unzip the archive with extension.
- Go to: Extensions -> Extension installer ->Upload your (filename.ocmod.zip) ocmod file.
- Go to: Extensions -> Modifications -> and click REFRESH button.
- Go to admin control panel -> Settings -> Server -> Live Chat -> Copy the code of Your Live Chat and insert it to the input area -> press Save button
- Done

for Opencart 1.5.x.x
- First You should to backup your opencart site and also database.
- Install VQmod. If not yet installed, visit http://support.tobiknet.com/knowledgebase.php?id=644b8cbc21e3404ab00bc10b8cfdec15
- Unzip the archive
- Copy *.xml file to ../vqmod/xml/ directory
- Go to admin control panel -> Settings -> Server -> Live Chat -> Copy the code of Your Live Chat and insert it to the input area -> press Save button
- Done

# Usage: Settings -> Server -> Live Chat -> Copy the code of Your Live Chat and insert it to the input area -> press Save button.
It works for code of "floating button" and other on-site overlay window.

!!! Works with ALL custom themes.
